# Code of Conduct

## Our standard

We are committed to a respectful, inclusive, and harassment-free community for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume good intent and discuss ideas, not people.
- Provide helpful feedback with clear technical context.

## Unacceptable behavior

- Harassment, discrimination, or hateful conduct.
- Personal attacks, intimidation, or doxxing.
- Deliberate disruption, trolling, or bad-faith engagement.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that violate this Code of Conduct.

## Reporting

Report conduct issues privately to the maintainers via GitHub security advisory contact or repository owner contact channels.

Maintainers will review reports promptly and take appropriate action.
